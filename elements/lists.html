Главная Элементы Списки

Маркированный список

  • Элемент маркированного списка
  • В маркированном списке порядок элементов не имеет значения. В качестве маркера используется квадрат. Это длинный элемент маркированного списка, который переносится на следующую строку.
  • Списки могут быть вложенными друг в друга:
    • Это элемент вложенного списка
    • Другой элемент вложенного списка
  • Последний элемент списка

Нумерованный список

  1. Элемент нумерованного списка
  2. В нумерованном списке элементы располагаются в заданном порядке. Это длинный элемент нумерованного списка, который переносится на следующую строку.
  3. Списки могут быть вложенными друг в друга:
    1. Первый элемент вложенного списка
    2. Второй элемент вложенного списка
  4. Последний элемент списка
